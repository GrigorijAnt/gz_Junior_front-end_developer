Git, привет!

Основные команды для адресной строки Git

cd url-адрес - перейти в папку c проектом

git init - создать с нуля новый репозиторий. Инициализация (папка .git)

git status - просмотреть список новых или изменёных файлов. Не закомиченых

git add . - Индексировать (подготовить) все файлы в ожидание коммита

git rm --cached имяФайла - убрать фаил из ожидания коммита

git commit -m "комментарий изменений" - Зафиксировать (сохранить) индексированные файлы с комментарием в историю

git log - показать полную историю изменений (коммитов)

git log --oneline - показать краткую историю изменений (коммитов)

git checkout id-коммита - откат к определённому коммиту по id, для ПРОСМОТРА! изменений

git checkout имяВетки - возврат к последней версии комитов ветки и переход по веткам

git revert id-коммита - удаление сделанных изменений в файлах определённого коммита по id, но не сам коммит

:wq - выход из редактора vin (непонятка)

git reset id-коммита - удаление коммитов выше указанного id-коммита (откат до указанной версии id), но сделанные изменения в файлах остаются

git reset id-коммита --hard - удаление комитов выше указанного id полностью, откат до указанной версии всё

gir branch имяВетки - создание новой ветки

git checkout имяВетки - переход по веткам

git checkout -b имяВетки - создание новой ветки с переходом на неё

git branch -a - позать список всех веток
